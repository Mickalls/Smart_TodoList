# 📋 Smart TodoList - 智能待办事项管理系统

一个基于Electron的现代化待办事项管理应用，集成了完整的AI智能功能套件，支持多AI提供商，提供优雅的用户体验和强大的任务管理能力。

## ✨ 功能特点

### 🎯 核心功能
- **多列任务管理**：今日任务、本周任务、长期目标、已完成任务的智能分类展示
- **优先级管理**：高、中、低三级优先级设置，支持视觉化优先级标识
- **回收站功能**：误删恢复机制，30天自动清理，防止数据丢失
- **多语言支持**：完整的中文/英文界面切换
- **主题切换**：明亮/暗黑模式，护眼体验
- **数据持久化**：本地存储，无需服务器，数据安全可靠
- **通知提醒**：到期任务自动提醒，支持系统通知
- **数据导出**：支持日报导出功能

### 🤖 AI增强功能套件

#### 1. 智能任务分类 🏷️
- **自动分析**：AI智能分析任务内容，推荐最合适的分类标签
- **多语言支持**：支持中英文任务内容分析
- **智能推荐**：基于任务描述自动推荐工作、学习、生活、其他等分类
- **用户决策**：AI推荐后用户可选择接受或手动调整

#### 2. 任务描述优化 ✨
- **智能优化**：AI分析并优化用户输入的任务描述
- **三种操作模式**：
  - 采用AI建议：直接使用AI优化的描述
  - 手动编辑：基于AI建议进行个性化修改
  - 忽略建议：保持原始描述不变
- **实时预览**：对比显示原始描述和AI优化建议

#### 3. 智能时间规划 📅
- **自动规划**：AI分析待办任务，生成智能时间安排
- **时间段分配**：按上午、下午、晚上三个时段智能分配任务
- **拖拽调整**：支持手动拖拽调整AI生成的时间安排
- **确认机制**：用户可确认或移除AI安排的任务时间

#### 4. 自然语言输入 💬
- **双模式输入**：
  - 📝 结构化模式：传统的表单输入方式
  - 💬 聊天模式：自然语言描述任务
- **智能解析**：AI自动解析自然语言，提取任务信息
- **信息提取**：从自然语言中提取任务内容、优先级、分类、时间等
- **确认编辑**：解析结果可编辑确认，确保准确性

### 🔧 AI服务支持
- **多AI提供商**：支持DeepSeek（推荐）、OpenAI、Claude、自定义API
- **智能配置**：一键配置AI服务，支持自定义API地址和模型
- **无缓存设计**：每次AI请求都实时调用，确保结果最新
- **优雅降级**：AI服务异常时自动使用默认处理方式
- **配置记忆**：自动保存用户AI配置，开箱即用

## 🚀 快速开始

### 环境要求
- Node.js 14.0+
- npm 6.0+

### 安装运行
```bash
# 克隆项目
git clone <repository-url>
cd <repository-name>

# 安装依赖
npm install

# 启动应用
npm start
```

### AI功能配置
1. 点击右上角 🤖 按钮打开AI配置
2. 选择AI提供商（推荐DeepSeek）
3. 配置API地址和模型
4. 填入API密钥
5. 启用相关AI功能
6. 保存配置

## 📖 使用指南

### 基本使用
1. **添加任务**：在输入框中输入任务描述
2. **设置优先级**：选择高、中、低优先级
3. **选择分类**：手动选择或使用AI智能分类
4. **管理任务**：点击完成、编辑、删除任务
5. **主题切换**：点击 🌞/🌙 切换明暗主题
6. **语言切换**：点击语言按钮切换中英文

### AI功能使用

#### 1. 智能任务分类
- 在分类下拉框中选择"🤖 智能分类"
- 输入任务描述后点击"添加"
- AI会自动分析并推荐分类
- 点击AI建议的标签接受，或选择其他分类

#### 2. 任务描述优化
- 输入任务描述后点击"✨ 优化"按钮
- AI会生成优化建议
- 选择"采用建议"、"手动编辑"或"忽略建议"
- 确认后任务将以优化后的描述保存

#### 3. 智能时间规划
- 点击"🧠 智能规划"按钮
- AI会分析当前待办任务，生成时间安排
- 可拖拽调整任务在不同时间段的安排
- 点击"确认安排"保存时间规划

#### 4. 自然语言输入
- 点击输入区域的"💬 聊天模式"标签
- 使用自然语言描述任务，如：
  - "明天下午3点提醒我开会"
  - "高优先级：完成项目报告"
  - "下周五前学习新技术"
- 点击"🤖 AI解析"按钮
- 在弹出的确认框中查看并编辑AI解析结果
- 点击"确认添加"保存任务

## 🔧 技术架构

### 前端技术栈
- **框架**：Electron 25.3.1
- **语言**：原生JavaScript（ES6+）
- **样式**：CSS3 + CSS Variables
- **存储**：localStorage
- **UI设计**：现代化扁平设计，响应式布局

### AI服务架构
- **配置管理**：`AIConfigManager` - 统一管理AI配置
- **API客户端**：`AIAPIClient` - 统一API调用接口
- **业务逻辑**：`AIService` - AI功能核心业务逻辑
- **UI交互**：完整的AI功能UI组件和交互逻辑
